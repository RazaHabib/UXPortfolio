Case Study

B2B MMS Administrator Redesign

Redesigned B2B MMS Administrator to control network security remotely with predictable, simple and beautiful interaction.

Role: Lead Product Manager
Year: 2023 (G DATA CyberDefence AG)
Platform: Web Application
Industry: B2B Cybersecurity

Project Overview

The B2B MMS (Mobile Management System) Administrator was a critical security tool that allowed IT administrators to remotely control and monitor network security across multiple endpoints. However, the existing interface was complex, unintuitive, and required extensive training for new users.

My challenge was to redesign this enterprise-level security application to make it more accessible while maintaining its powerful functionality. The goal was to create a web-based product with progressive complexity revelation that would allow both novice and expert users to efficiently manage network security.

My Role & Responsibilities

  • **Product Leadership:** Led the product vision, strategy, and roadmap for the B2B MMS Administrator redesign, aligning with business objectives and user needs.
  • **UX Strategy & Research:** Conducted extensive user research, competitive analysis, and user interviews to identify pain points and inform design decisions.
  • **Information Architecture & Interaction Design:** Restructured navigation and content hierarchy, implementing progressive disclosure to manage complexity for diverse user groups.
  • **Cross-Functional Team Management:** Managed a cross-functional team of designers, developers, and QA to deliver the redesigned application.
  • **Performance & Impact Measurement:** Defined and tracked key performance indicators (KPIs) to measure the success of the redesign, focusing on user efficiency and satisfaction.

Key Challenges

  • Overcoming a complex legacy interface with a steep learning curve for new IT administrators.
  • Maintaining powerful enterprise functionality while simplifying the user experience.
  • Designing for both novice and expert users with varying technical proficiencies.
  • Ensuring robust security controls were intuitively accessible and manageable.

Solutions & Impact

Implemented a comprehensive dashboard for at-a-glance security status and customizable widgets. Introduced progressive complexity, showing basic controls by default and advanced features through contextual menus. Enhanced security monitoring with improved alerts and data visualization.

The redesign led to a **40% improvement in user efficiency**, a **60% reduction in task completion time**, and an **85% user satisfaction score**. New user onboarding time decreased from 2-3 weeks to less than 1 week.
