B2B MMS Administrator Redesign

Redesigned B2B MMS Administrator to control network security remotely with predictable, simple and beautiful interaction

Role Senior Product Manager & UX Lead
Company G DATA CyberDefence AG
Duration 6 months
Team Cross-functional (5 members)

Project Overview

As Senior Product Manager at G DATA CyberDefence AG, I led the complete redesign of the B2B MMS (Mobile Management System) Administrator interface. This critical security management tool needed to evolve from a complex, technical interface to an intuitive, user-friendly platform that could be operated remotely with confidence.

Key Achievement

40% improvement in user efficiency through streamlined workflows and predictable interaction patterns.

The Challenge

The existing MMS Administrator interface suffered from several critical usability issues:

  • Complex Navigation: Users struggled to find essential security controls
  • Inconsistent Patterns: Different sections used varying interaction models
  • Remote Operation Difficulties: Interface wasn't optimized for remote management scenarios
  • Information Overload: Critical security alerts were buried in technical details

Research & Discovery

I conducted comprehensive user research to understand the needs of security administrators and IT professionals who rely on remote network management.

User Interviews

15 security administrators across different company sizes

Usability Testing

Task-based testing of critical security workflows

Competitive Analysis

Analysis of leading B2B security management platforms

Analytics Review

Deep dive into user behavior and pain point identification

Design Process

Following a user-centered design approach, I developed a comprehensive redesign strategy focusing on predictability, simplicity, and remote operation efficiency.

Information Architecture

Restructured the entire navigation system based on user mental models and task frequency analysis.

Wireframing & Prototyping

Created low-fidelity wireframes to test core concepts, then developed interactive prototypes for validation.

Final Design Solution

The redesigned MMS Administrator interface features a clean, predictable layout optimized for both local and remote security management tasks.

Key Design Features

  • Unified Dashboard: Critical security metrics at a glance
  • Contextual Actions: Relevant controls appear based on current context
  • Progressive Disclosure: Complex settings revealed only when needed
  • Status Indicators: Clear visual feedback for all system states

Key Learnings

  • Context is King: B2B security tools must adapt to high-stress, time-critical scenarios
  • Progressive Disclosure: Complex enterprise software benefits from layered information architecture
  • Remote-First Design: Modern B2B tools must be optimized for distributed teams and remote work
  • Stakeholder Alignment: Regular cross-functional collaboration ensures technical feasibility and business alignment
