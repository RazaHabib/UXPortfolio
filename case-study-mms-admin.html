B2B MMS Administrator Redesign

Redesigned B2B MMS Administrator to control network security remotely with predictable, simple and beautiful interaction

Role Lead UX Designer
Year 2023
Platform Web Application
Industry B2B Security
40% Improved User Efficiency
60% Reduced Task Completion Time
85% User Satisfaction Score

Project Overview

The B2B MMS (Mobile Management System) Administrator was a critical security tool that allowed IT administrators to remotely control and monitor network security across multiple endpoints. However, the existing interface was complex, unintuitive, and required extensive training for new users.

My challenge was to redesign this enterprise-level security application to make it more accessible while maintaining its powerful functionality. The goal was to create a web-based product with progressive complexity revelation that would allow both novice and expert users to efficiently manage network security.

The Challenge

  • Complex legacy interface
  • High learning curve for new users
  • Inefficient task completion workflows
  • Poor visual hierarchy
  • Limited mobile responsiveness

Research & Discovery

User Interviews

Conducted interviews with 12 IT administrators to understand their daily workflows, pain points, and feature priorities.

Competitive Analysis

Analyzed 8 competing security management platforms to identify industry best practices and opportunities for differentiation.

Key Insights

1

Users spent 70% of their time on routine monitoring tasks that could be automated or simplified

2

Critical security alerts were often missed due to poor visual hierarchy and notification systems

3

New team members required 2-3 weeks of training to become proficient with the existing system

Design Process

Information Architecture

Restructured the navigation and content hierarchy based on user task frequency and importance. Created a progressive disclosure system that reveals advanced features as needed.

Wireframing & Prototyping

Developed low-fidelity wireframes focusing on layout optimization and user flow improvements. Created interactive prototypes to test key user journeys.

Visual Design

Applied a clean, modern design system with improved color coding for security status, better typography hierarchy, and intuitive iconography.

The Solution

Dashboard Redesign

Created a comprehensive dashboard that provides at-a-glance security status with customizable widgets and real-time monitoring capabilities.

Progressive Complexity

Implemented a layered interface that shows basic controls by default, with advanced features accessible through contextual menus and expandable sections.

Enhanced Security Monitoring

Redesigned the security monitoring interface with improved alert systems, better data visualization, and streamlined incident response workflows.

Results & Impact

40%

Improved Efficiency

Users completed routine tasks 40% faster with the new interface design and streamlined workflows.

60%

Reduced Training Time

New user onboarding time decreased from 2-3 weeks to less than 1 week with improved usability.

85%

User Satisfaction

Post-launch user satisfaction scores increased significantly, with users praising the intuitive design.

25%

Reduced Errors

Configuration errors decreased by 25% due to improved visual feedback and validation systems.

Lessons Learned

Progressive Disclosure is Key

For complex enterprise applications, progressive disclosure allows users to access advanced features without overwhelming newcomers. This approach significantly improved both novice and expert user experiences.

Security UX Requires Special Consideration

Security interfaces need to balance ease of use with the critical nature of the tasks. Clear visual indicators and confirmation dialogs for high-risk actions are essential.

Stakeholder Alignment is Crucial

Regular check-ins with both technical stakeholders and end users throughout the design process ensured the solution met both business requirements and user needs.
